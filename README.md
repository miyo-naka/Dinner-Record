- 開発環境：http://localhost:3000

## 関連レポジトリ

https://github.com/miyo-naka/Dinner-Record

## 機能一覧

- ログイン・ユーザー登録
- 記録作成
- 記録一覧表示
- カレンダー表示切替
- 日付ソート機能
- カテゴリーによる絞り込み機能
- 記録編集
- 記録削除
  <br><br>

## 使用技術(実行環境)
- next.js 15.1.7
- react 19.0.0
- tailwindcss 3.4.1
- firebase 11.3.1
  <br><br>

## Getting Started

### 1. リポジトリのクローン
```bash
git clone リポジトリのURL プロジェクト名
cd プロジェクト名
```

### 2. 環境構築
サーバーの起動:
```bash
npm run dev
```
